# Danone Internship Case Study: Análisis de Ventas en España

## Cómo Ejecutar el Proyecto (Instrucciones)

Para replicar y ejecutar el análisis, sigue estos pasos, asegurándote de trabajar en un entorno virtual aislado.

### 1. Requisitos del Entorno (Python)

El proyecto requiere **Python 3.8+** y un entorno virtual para gestionar las dependencias.

#### A. Crear y Activar el Entorno Virtual

Abre tu terminal o símbolo del sistema y ejecuta:

```bash
# Crea el entorno llamado 'danone_env'
python3 -m venv danone_env 

# Activa el entorno
# En macOS/Linux:
source danone_env/bin/activate
# En Windows (CMD o PowerShell):
.\danone_env\Scripts\activate
